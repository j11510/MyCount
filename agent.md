엑셀로 관리하던 지출내역계산을 웹에서 등록할수 있게 페이지를 만들고 싶어
화면 구조는 첫번째 항목은 마이너스 고정금액 그리고 
두번째 항목은 현재 통장 잔고 입력 세번째 항목은 고정금액 - 현재 통장 잔고 
그리고 아래는 고정금액 - 현재통장 잔고에서 월지출항목을 나열하고 항목별로 
금액을 입력할수 있게 하고 항목을 추가 및 삭제도 가능하게 하고 제일 
아래는 남은 잔고에 항목별 금액을 뺀 결과 금액을 표시
그리고 현재 계산내역을 DB에 저장해서 이력확인 할수 있게 월별로
화면 옆에는 최종 결과: 300,000원 에서 수입을 항목별로 추가 삭제하여 
최종잔고 수입을 계산 그래서 내가 알고 싶은건 이달에 얼마가 필요한 금액인지 계산하고 싶어 
지출항목중에서 고정을 선택하면 항목이 고정되고 신규로 등록할때도 고정된 지출항목이 나와 됨
고정항목도 내가 금액을 1회성으로 수정가능해야 해 그리고 다시 다음달 신규로 등록할때는 이전에 
고정금액으로 그리고 지출 고정항목들은 별도로 추가 및 삭제를 관리하고 싶어 항목 추가, 금액변경 등
이 화면 메뉴들은 관리자 권한만 볼수 있어야 됨 메뉴명 자체도 일반 유저는 볼수 없게

개발 환경은 python + flask + fastapi + nextjs + 리액트 + 마리나db
